st - simple terminal
--------------------
st is a simple terminal emulator for X which sucks less.


Features pre loaded
------------
```
1. Scroll
2. Transparent Background
3. Fonts support 
```

Installation
------------
```
paru -S st-hackme
or
yay -S st-hackme
```
Running st
----------
Search in dmenu or rofi as or simply type it in terminal
```
st
```
